=== Description ===

1/ Télécharger le dossier de l'extension dans le repertoire 
wp-content/plugins.
2/ Activez l'extension.
3/ Cliquez sur Info-Pack
4/ Allez à l'onglet My IP, copie le shortcode et colle le dans l'une des page de ton site.
5/ Allez à l'onglet My POST, copie le premier ou le deuxième shortcode et colle le dans l'une des page de ton site.
6/ Le dernier onglet My PLAN se met dans une page dédié au plan de sie. copie le shortcode et colle le dans la page adéquate de ton site.
7/ L'onglet du menu support va te servir à demander de l'aide en cas de prolème avec le plugin
